# George Hanna Portfolio

Static portfolio for GitHub Pages.

## Edit Content

- Main content and translations: `app.js`
- Layout: `index.html`
- Visual design: `styles.css`
- Resume download: `assets/resume/george-hanna-cv.pdf`
- Future 3D models: place `.glb` files in `assets/models/` and add their paths to the matching project in `app.js`.

## GitHub Pages

1. Push this repository to `georgehanna1993/portfolio`.
2. In GitHub, open **Settings > Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Select the `main` branch and root folder `/`.
5. The site will be available at `https://georgehanna1993.github.io/portfolio/`.

The contact form uses FormSubmit and sends to `georgehanna1993@gmail.com`. The first submission may require email confirmation from FormSubmit.
